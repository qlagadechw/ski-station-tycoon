# 🎿 Ski Station Tycoon

Un jeu de gestion de station de ski en 3D, développé avec Godot 4.

## Concept

Construisez et gérez votre propre station de ski dans le **Massif du Jura** !

- 🏔️ Terrain 3D procédural avec vallées et montagnes
- ⛷️ Construisez des pistes de ski (verte, bleue, rouge, noire)
- 🚡 Installez des remontées mécaniques (téléski, télésiège, télécabine, téléphérique)
- 🔵 Reliez votre réseau avec des hubs de connexion
- 💰 Gérez vos finances (forfaits, entretien, construction)
- 🕐 Saisons dynamiques avec enneigement réaliste
- 🤖 Skieurs IA avec comportements réalistes

## Technologie

- **Moteur** : Godot 4.x
- **Langage** : GDScript
- **Plateforme cible** : Steam (Windows, Linux, macOS)
